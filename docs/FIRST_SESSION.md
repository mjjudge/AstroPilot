# First Session Plan (At Home)

## Goal
Get one motor moving under controlled input

## Steps
1. Photograph RA motor label
2. Count wires and identify connector
3. Identify spare stepper driver model
4. Wire RA motor to driver (power OFF)
5. Connect driver to ESP32
6. Run basic step test
7. Verify smooth rotation and direction

## Success Criteria
- Motor turns smoothly
- No overheating
- Direction controllable

## Do Not
- Power motors from USB
- Hot-plug anything
