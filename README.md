# AstroPilot: EQ4 Smart Mount Retrofit

Goal: Convert existing EQ4 mount into a GoTo-capable system using ESP32 (OnStepX), Raspberry Pi, and iPad UI.

## Architecture
- ESP32: real-time motor control (OnStepX)
- Raspberry Pi: web UI + orchestration
- iPad: browser-based control panel

## Current Phase
Phase 0 — Discovery (motor + driver validation)

## Next Step
Identify motor specs and confirm driver compatibility.
